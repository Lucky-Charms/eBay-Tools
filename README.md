# eBay-Tools
Program that can generate views, watches, and ebay accounts for a specific product.


# Current Features
- Mass Create Accounts using Catch-all domain.
- Add X Watchers to a listing.
- Add X Amount of Views to a Listing.


# Require
- Electron
- Selenium
- Chrome Driver
- Electron Settings
